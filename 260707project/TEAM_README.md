# SMILE — 행동 교정 맞춤 동화 생성 서비스

## 1. 프로젝트 소개

SMILE은 아이의 **문제 상황**과 **성격**을 입력하면, 올바른 행동을 자연스럽게 유도하는 **맞춤형 그림동화**를 AI가 생성해주는 서비스입니다.

- **한 줄 소개**: 아이의 문제 행동을 교정하는, 우리 아이 맞춤형 AI 그림동화 생성 서비스
- **주요 사용자**: 만 3~7세 영유아 자녀를 둔 부모·양육자
- **만들게 된 배경**: 훈육 과정에서 부모는 "하지 마"식 지시나 "안 그러면 무서운 사람이 잡아간다"식 정서적 협박에 의존하기 쉽습니다. 전문가 지침이 있어도 우리 아이 상황에 맞게 적용하기는 어렵습니다. 아이는 지시보다 **이야기에 몰입**하므로, 아이를 주인공으로 한 맞춤 동화가 효과적인 교정 도구가 될 수 있습니다.
- **최종 결과물의 형태**: 동화 생성·뷰어 웹앱(프론트엔드) + 동화 생성 API 서버(백엔드)

> SMILE은 아이의 감정을 100% 수용하고, 놀이식 시뮬레이션을 통해 긍정적인 방향으로 행동을 교정하도록 설계된 AI 동화 생성 서비스입니다.

---

## 2. 문제 정의

- **사용자가 겪는 불편함**: 부모는 편식·때리기·떼쓰기·양치 거부 같은 아이의 문제 행동 앞에서 효과적인 대응법을 몰라 지시나 협박에 의존하게 되고, 이는 아이에게 공포·죄책감을 심을 수 있습니다.
- **기존 방식의 한계**:
  - 시중 동화는 **우리 아이의 특정 문제·성향에 맞춰져 있지 않음**
  - 전문가 지침(보건복지부 등)은 존재하지만 **딱딱하고, 개별 상황에 적용하기 어려움**
  - 맞춤 동화를 매번 사람이 만드는 것은 현실적으로 불가능
- **왜 중요한가**: 영유아기의 행동 교정은 **어떻게 접근하느냐**가 결정적입니다. 공포·협박이 아니라 공감과 긍정적 대안 제시가 아이의 정서 발달에 큰 영향을 줍니다.

## 3. 문제 해결

- **핵심 아이디어**: 보건복지부 전문가 지침의 **'심리학적 원칙'만 추출**해, 아이의 상황·배경에 맞는 **동화로 재창작**합니다. (지침의 구체적 소품을 그대로 옮기지 않고 원칙만 이식 → 어떤 이야기 세계관에도 자연스럽게 녹아듦)
- **AI/소프트웨어 활용 방식**:
  - **RAG(검색 증강 생성)**: 입력된 문제 행동으로 보건복지부 지침(ChromaDB)을 검색해 해결 원칙을 확보
  - **LangGraph 멀티에이전트**: `검색 → 초안 → 자동 검수` 루프로 동화를 생성하고, 금지 표현·분량·**결말 실천(회피형 결말 차단)**을 자동 검증
  - **이미지 생성**: 페이지별 프롬프트로 일관된 화풍의 삽화 생성
- **전체 동작 흐름**:

```
아이 정보 + 문제 상황·분위기 입력
  → 전문가 지침 RAG 검색 (해결 원칙)
  → LLM 멀티에이전트 동화 생성 (자동 검수 루프)
  → 사용자 초안 검토 → 피드백 수정(첨삭) → 확정
  → 페이지별 삽화 생성 → 완성된 그림동화
  → 별점 4점↑ 동화는 예시로 재활용 (품질 선순환)
```

> 자세한 아키텍처·플로우는 [`FLOWCHART.md`](FLOWCHART.md), RAG 검색 성능 개선 과정은 [`WALKTHROUGH.md`](WALKTHROUGH.md)를 참고하세요.

## 4. 핵심 기능

- **맞춤형 동화 생성** — 아이의 이름·나이·성별·성격·문제 상황·배경을 반영한 기승전결 4페이지 동화
- **RAG 기반 전문가 지침 적용** — 보건복지부 지침의 원칙을 이야기로 재창작 (정서적 협박·공포 배제)
- **멀티에이전트 자동 검수** — 금지 표현·분량·배경 반영에 더해, 결말에서 목표 행동을 실제로 수행했는지(회피형 결말 차단)까지 LLM으로 검사
- **Human-in-the-Loop 첨삭** — 초안 확인 → 피드백으로 수정 → 확정의 3단계
- **분위기 기반 예시 매칭 + 피드백 선순환** — 사용자 평점 4점↑ 동화를 예시 풀로 승격해 품질을 지속 개선
- **부모용 육아팁 제공** — 동화에 쓰인 전문가 지침을 다정한 팁으로 요약 제공
- **삽화 자동 생성** — 페이지별 일관된 화풍의 그림
- **한국어 / 영어 지원**
