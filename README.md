# Children's Story Generator

AI 맞춤형 행동 교정 동화 서비스의 초기 MVP입니다.

현재 버전은 API를 호출하지 않습니다. 부모가 입력한 동화 요구사항을 바탕으로 LLM에 넣을 최종 프롬프트를 미리보기로 생성합니다.

## 포함된 기능

- 아이 기본 정보 입력
- 주인공 외형 및 캐릭터 특징 입력
- 문제 상황, 교훈, 분위기, 배경 입력
- 동화 생성용 프롬프트 미리보기

## 실행 준비

Next.js 실행에는 Node.js 18 이상이 필요합니다.

```bash
node -v
```

Node 버전이 18보다 낮다면 먼저 Node를 업데이트해 주세요.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 아래 주소를 열면 됩니다.

```text
http://localhost:3000
```
